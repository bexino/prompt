```
# 图像修复

## 1. 核心目标 (Core Objective)
将上传的图像升级为一个纯净无瑕、超高清、电影感版本。在去除所有物理损伤（残损、模糊、斑驳、磨花）并完成黑白上色的同时，以绝对保真度保留主体。

## 2. 绝对保真要求 (Strict Fidelity Requirements)
人物的身份(Identity)、面部解剖结构(Facial Anatomy)、表情(Expressions)、身体姿态(Body Posture)、服装(Clothing)、配饰(Accessories)、环境(Environment)、取景框定(Framing)以及整体构图(Composition)都必须保持完全不变。严禁修改、重新诠释、替换，或引入任何新的视觉元素。

## 3. 分阶段修复与上色工作流 (Step-by-Step Workflow)

### 阶段一：物理修复与结构重建 (Physical Restoration)
* **消除残损与斑驳**：精准识别并消除图像上的所有撕裂、破损、霉点、污渍以及斑驳痕迹。
* **磨花与划痕修复**：无痕修复画面中的划痕、摩擦痕迹与物理磨损，根据周围合理的纹理和结构进行无缝补全。
* **边界完整性**：确保图像边缘和主体轮廓在修复后保持绝对的几何正确性。

### 阶段二：黑白上色与色彩保真 (Colorization)
* **真实感上色**：基于历史准确性与现实逻辑进行分层上色。拒绝高饱和度或现代感过强的滤镜色彩。
* **精准肤色与材质**：赋予皮肤自然健康的血色与光泽，精准还原服装（如制服、呢料、丝绸）与环境建筑的原本材质色彩。
* **色彩和谐度**：确保上色后的整体色调自然融合，服务于电影感的视觉呈现。

### 阶段三：去模糊与微观细节重建 (De-blurring & Micro-details)
* **高级去模糊**：消除镜头模糊、运动模糊或年代原因导致的画面解析力不足，使图像边缘干净、清晰。
* **微观细节重建**：
  (a) 精确的面部轮廓与清晰的五官线条。
  (b) 具有自然可见毛孔、细腻且富有呼吸感的真实皮肤纹理，严禁出现塑料感或过度磨皮。
  (c) 根根分明、质感自然的丝丝发丝。
  (d) 清晰而逼真的眼睛，包含解析充分的虹膜细节与自然的瞳孔高光。
  (e) 贯穿整张图像的干净、清晰、解析充分的边缘。

### 阶段四：灯光与画面电影感增强 (Cinematic Lighting)
* **影棚级光影**：引入平衡的、影棚级的电影感灯光，重新校准画面的动态范围(Dynamic Range)与对比度。
* **空间纵深感**：通过细腻的加深与减淡(Dodge & Burn)增强主体的立体感，拉开主体与背景的微弱空间层次。
* **最终调色**：在严格融入原图环境的基础上进行微妙的电影感色彩分级，使最终图像呈现出纯净、高级、兼具历史厚重感与现代超高清质感的视觉特效。
```
